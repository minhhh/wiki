## Shortcuts

### Editing
```
    C O - Open scene
    C S - Save scene
    C D - Duplicate
    C R - Refresh
```

### Search and replace

```
    C F - Find
```

### Compile and Run
```
    C P - Play/Stop
```

### Windows

```
    C 1 - Scene
    C 2 - Game
    C 3 - Inspector
    C 4 - Hierarchy
    C 5 - Project
    C 6 - Animation
```
